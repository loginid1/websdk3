# `@loginid/core`

> Shared internal code used across all LoginID SDKs

This package includes:

- API clients for the FIDO2 service
- Shared stores, helpers, and utility functions
- Reusable validators, controllers, and error types
- WebAuthn helpers for client-side passkey handling using WebAuthn APIs

## Internal Package

This is an _internal package_ to the [LoginID SDK monorepo](https://gitlab.com/loginid/software/sdks/loginid-websdk3).  
It is not meant to be used directly outside of this monorepo.
