The LoginID Checkout SDK for Merchants enables seamless passwordless authentication using passkeys within embedded 
checkout experiences. This SDK is designed to integrate with iframes and orchestrate user authentication in a secure way.

## Features

- **Passkey Authentication Integration**: Enables merchants to embed passwordless authentication directly into their checkout flows.
- **Embedded Orchestration**: Supports secure iframe-based communication between the merchant frontend and the authentication service.
- **Cross-Origin Support**: Built with modern web constraints in mind, allowing safe and efficient communication between different application origins.

## Installation

To integrate the LoginID Checkout Merchant SDK into your project, run:

    npm install @loginid/checkout-merchant
