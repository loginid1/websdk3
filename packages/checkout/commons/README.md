# `@loginid/checkout-commons`

> Shared internal code for the LoginID Checkout SDKs

This package includes:

- Shared constants used across Checkout SDKs
- Common TypeScript typings
- Utility and helper functions for both Merchant and Wallet SDKs

## Internal Package

This is an _internal package_ to the [LoginID SDK monorepo](https://gitlab.com/loginid/software/sdks/loginid-websdk3).  
It is not intended for direct external use.
